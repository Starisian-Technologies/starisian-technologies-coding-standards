# Support

## Usage questions

For standards usage questions, open a GitHub issue using the documentation template.

## Defects and regressions

Open a GitHub issue with reproducible steps, expected behavior, actual behavior, and environment details.

## Security reports

Do not disclose security vulnerabilities publicly. Follow `SECURITY.md` for private reporting.

## Response expectations

- Initial triage target: within 3 business days
- Security reports: prioritized triage on receipt
- Standards clarifications: resolved through issue discussion and linked governance references
